# **App Name**: Agent Finder Pro

## Core Features:

- Lead Import & Intake Workflow: Streamline the import of scraped real estate agent leads from external sources, providing tools to review, clean, deduplicate, and triage records efficiently before approval into the main CRM.
- AI Lead Classification & Scoring Tool: Utilize an AI-powered tool to automatically analyze lead data, assigning an 'independent_score' and classifying agents as 'likely_independent', 'possible_independent', or 'agency' with a transparent explanation for the assessment.
- Approved Leads Management: Centralized table view for all approved real estate agent leads, offering powerful search, filtering, sorting, customizable views, bulk actions, and visual cues to prioritize independent agents.
- Lead Detail View & Activity Timeline: A comprehensive, polished interface for individual lead details, including contact information, independent score explanation, notes, tasks, and a unified cross-channel activity timeline of all interactions.
- Multi-Channel Outreach Campaigns: Configure and manage automated multi-step outreach sequences for email, WhatsApp, and SMS, allowing personalization and defining rules for progression based on lead interaction, integrating with external sending services.
- Dynamic Outreach Template Editor: Create, store, and manage reusable outreach templates for various channels (email, WhatsApp, SMS) with support for variables, enabling personalized communication and dynamic content preview.
- Sales Performance Dashboard: A visually impressive dashboard presenting key performance indicators related to lead intake, agent classification, outreach campaign efficacy, task management, and overall conversion rates in a clean layout.

## Style Guidelines:

- Primary brand color: A refined, muted blue (#51648F) conveying trust and professionalism against a light backdrop.
- Background color: A very pale blue-gray (#EEF1F7), almost white, providing an airy and clean canvas for content.
- Accent color: An elegant violet-blue (#8C6FDC), used sparingly for interactive elements, highlights, and calls to action to add a premium touch.
- Headlines and key text elements will use 'Space Grotesk' (sans-serif) for a modern, technically precise aesthetic. Body text, table content, and general readability will utilize 'Inter' (sans-serif).
- Utilize a consistent set of refined, line-art or glyph-style icons. These should be minimalist to support the clean and uncluttered visual goal, providing clarity without adding visual weight.
- Employ a desktop-first, airy layout with generous use of white space to enhance scanability and visual hierarchy. Key information will be presented in modern, rounded-corner cards with soft shadows, accompanied by highly readable tables and polished drawer panels for detail views.
- Incorporate subtle and smooth hover states for interactive elements, gentle transitions for navigation, and minimal, non-distracting animations for data loading or state changes, prioritizing operational efficiency.
